# CoffeeTech Decoded ‚òï

**Interactive 3D visualization platform for coffee brewing methods.**

CoffeeTech Decoded is an educational web application that breaks down the mechanics of popular coffee equipment using interactive 3D models. Built with React and Three.js, it offers exploded views, brewing animations, and technical specifications for enthusiasts and learners alike.

## üöÄ Features

- **Interactive 3D Models**: Rotate, zoom, and explore detailed equipment models.
- **Exploded Views**: Visualize internal components with a single click.
- **Brewing Animations**: Watch step-by-step simulations of the brewing process (e.g., water flow, pressure extraction).
- **Technical Specs**: View detailed parameters like pressure, temperature, and grind size.
- **Performance Radar**: Visual comparison of flavor profiles (Body, Acidity, Complexity, etc.).
